# Suggested Commands
## Development
- `npm run dev` : Run Next.js dev server (typically at `http://localhost:3000`).
- `npm run build` : Build production bundle.
- `npm run start` : Start production server from built assets.

## Quality
- `npm run lint` : Run ESLint checks.

## Useful macOS/Darwin shell commands
- `ls -la` : List files with details.
- `find . -maxdepth 2 -type d` : Inspect directory tree quickly.
- `rg --files` : Fast file listing/search.
- `rg "pattern" app` : Search code/text quickly.
- `git status` : Check repository state.
- `git diff` : Inspect local changes.
